# 🍔 BurgerThing Interactive E-Kiosk

An interactive, touchscreen-optimized food ordering e-kiosk simulation built entirely in Python using **Streamlit**. This application mimics the modern self-service kiosks found in major fast-food chains, providing a smooth user experience from menu browsing to digital checkout.

[![Open in Streamlit](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://blank-app-qla9u0z7hn.streamlit.app/)

---

## 🚀 Live Demo & Concept
This project showcases how Python can be used to build fast, reactive, and highly visual front-end interfaces without needing complex JavaScript frameworks. It features high-fidelity visual assets, real-time cart math, and a responsive layout designed to look like an in-store touch monitor.

## ✨ Core Features
*   **📱 Kiosk-Optimized Layout:** Dual-column design featuring an intuitive menu browser on the left and a persistent digital tray/receipt on the right.
*   **🍔 Dynamic Tabbed Menu:** Fast navigation across distinct food categories (Burgers, Sides, Drinks & Sweets) with crisp, real-time image rendering.
*   **🛒 Reactive Cart System:** Seamlessly add items, increment/decrement quantities, and remove products dynamically.
*   **🧮 Auto-Tax & Totaling:** Instant calculations tracking food subtotals, regional sales taxes (8%), and final totals on every click.
*   **💳 Simulated Touch Checkout:** Complete order processing featuring custom UI alerts, micro-animations, and a randomized customer collection ticket number.

---

## 🛠️ Technical Stack
*   **Language:** Python 3.9+
*   **Framework:** Streamlit (State-driven UI components)
*   **Styling:** Embedded CSS (Custom container weights, font styles, and touch-button scaling)
*   **Assets:** Dynamic CDN image routing via Unsplash
